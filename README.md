EduSnack – Projeto Integrado em Engenharia de Software III
## Descrição

O EduSnack é um aplicativo mobile voltado para escolas, criado para facilitar a organização da cantina.
Ele permite que cardápios sejam cadastrados e visualizados, pedidos sejam feitos com antecedência e retirados de forma rápida, reduzindo filas e melhorando o fluxo no intervalo escolar.

##Objetivo

Oferecer uma solução prática que ajude a cantina, alunos e responsáveis a gerenciarem pedidos de merenda com mais agilidade, transparência e segurança.
